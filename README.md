## üí° What is PulseDeck Presenter?
PulseDeck Presenter is an **Adobe Express add-on** that turns a brief, URL, or brand kit into a structured, on-brand presentation in under **2 minutes**.

### How it Works
1. **Input:** User pastes a text brief, URL, or meeting notes.
2. **Analyze:** AI analyzes the content to extract key points and structure.
3. **Generate:** Creates a full deck outline with suggested layouts.
4. **Refine:** Opens directly in Adobe Express for final editing with full creative control.

### Key Differentiator
Seamlessly integrates with **Adobe Express**, automatically applying your **brand kit** (colors, fonts, logos) while giving creators full editing capabilities‚Äîunlike other "black box" AI tools.

## üöÄ Getting Started

### Installation (Development)
1. **Clone the repository:**
   ```bash
   git clone https://github.com/astrix-05/PulseDeck.git
   cd PulseDeck/pulsedeck-landing
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file in the root directory:
   ```env
   GMAIL_EMAIL=your-email@gmail.com
   GMAIL_PASSWORD=your-app-password
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the landing page.
